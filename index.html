1

Wat is vibecoding?

Stel je voor: jij beschrijft in gewone, alledaagse taal wat je wilt maken — en een AI (Claude) schrijft alle code voor je. Jij bent de regisseur, Claude is de programmeur. Dat is vibecoding in een notendop.

Waarom dit een gamechanger is

Je hebt geen programmeerkennis nodig. Helemaal niks. Wat je wél nodig hebt: een idee en de vaardigheid om dat helder te beschrijven. Kun je uitleggen wat je wilt aan een collega? Dan kun je vibecoden.

Superpowers: jouw geheime wapen

Superpowers zijn een gestructureerde manier om met Claude te werken. In plaats van zomaar wat te vragen, werk je in drie stappen:

  1. Brainstormen — Wat wil ik eigenlijk? Claude helpt je jouw vage idee scherp te maken.
  2. Plannen — Hoe gaan we het bouwen? Claude maakt een blauwdruk.
  3. Bouwen — Claude schrijft de code, jij kijkt mee en stuurt bij.
Denk eraan als een architect: eerst schets je op een bierviltje, dan maak je een bouwtekening, en dán ga je pas bouwen. Superpowers zorgen ervoor dat je die stappen niet overslaat.
2

Wat heb je nodig? (Setup)

Voordat we aan de slag gaan, moeten we een paar dingen installeren. Geen paniek, vriend — ik loop elke stap met je door!

Stap 1: Claude Code installeren

Ga naar de officiële downloadpagina van Claude Code en installeer het programma. Na installatie heb je ook een klein tegoed nodig (tokens) om Claude te laten werken.

Tip: Start met het kleinste bundeltje. Als je enthousiast wordt (en dat ga je worden!) kun je later altijd upgraden naar een groter pakket.
PowerShell (als Administrator)
> winget install Anthropic.Claude
Installatie gelukt!

Of download het installatieprogramma van claude.ai/download en dubbelklik erop.

Terminal
$ brew install claude
Claude Code is geïnstalleerd!

Geen Homebrew? Download het installatieprogramma van claude.ai/download.

Stap 2: GitHub account aanmaken

Ga naar github.com en maak een gratis account aan. GitHub is waar je code opslaat én waar je site straks live gaat — zichtbaar voor de hele wereld!

Onthoud je gebruikersnaam en wachtwoord goed. Die heb je later nodig.

Stap 3: Een werkmap aanmaken

Dit wordt de plek waar je alles neerzet: foto's, teksten, inspiratieplaatjes. Claude “ziet” alles wat in deze map staat en kan het gebruiken.

PowerShell
> mkdir C:\mijn-project
Directory: C:\mijn-project aangemaakt

Of maak gewoon een nieuwe map aan via Verkenner. Waar je wilt, bijvoorbeeld op je bureaublad.

Terminal
$ mkdir ~/mijn-project

Of maak een nieuwe map aan via Finder.

Pro-tip: Zet alvast wat materiaal in je werkmap! Een paar foto's, een Word-document met teksten, een Excel — alles wat je wilt gebruiken. Claude kan het allemaal zien en ermee werken.

Stap 4: Claude Code starten

Nu wordt het spannend! Open een terminal, navigeer naar je werkmap, en start Claude.

PowerShell
> cd C:\mijn-project
> claude

  Claude Code v1.x
  Welkom! Ik ben klaar om je te helpen.
  Typ je vraag of opdracht...
Terminal
$ cd ~/mijn-project
$ claude

  Claude Code v1.x
  Welkom! Ik ben klaar om je te helpen.
  Typ je vraag of opdracht...

Je ziet nu een prompt waar je kunt typen. Dit is het! Je bent klaar om te vibecoden.

3

Je eerste gesprek met Claude

Claude werkt als een gesprek — je typt gewoon in het Nederlands wat je wilt, en Claude antwoordt. Geen speciale commando's, geen ingewikkelde syntax. Gewoon praten.

Je allereerste opdracht

Laten we simpel beginnen. Typ dit:

Claude Code
> Maak een HTML-pagina met de tekst Hallo Wereld

Claude gaat nu code schrijven en slaat het op als een bestand in je werkmap. Je zult iets zien als:

Claude's reactie
Ik maak een HTML-pagina voor je aan...

✓ Bestand aangemaakt: index.html

Je kunt het bekijken door index.html te openen
in je browser (dubbelklik op het bestand).

Dat is het! Open het bestand in je browser en je ziet je eerste webpagina. Vriend, je bent officieel een vibecoder!

Referentiemateriaal gebruiken

Heb je bestanden in je werkmap gezet? Claude kan ze gebruiken! Zeg bijvoorbeeld:

Claude Code
> Gebruik de foto's in mijn map om een fotogalerij te maken

Permissions: Claude vraagt toestemming

Claude vraagt soms of hij iets mag doen (een bestand aanmaken, iets installeren). Dit zijn je opties:

Allow
Ja, doe het. Eenmalige toestemming voor deze actie.
Allow always
Ja, en vraag het niet meer voor dit soort acties.
Deny
Nee, doe dit niet.
Dangerous mode
Claude doet alles zonder te vragen. Handig, maar let op: Claude kan dan ook bestanden verwijderen of overschrijven!
Advies voor beginners: Gebruik gewoon “Allow” in het begin. Dan zie je precies wat Claude doet en leer je het proces kennen. Later, als je meer vertrouwen hebt, kun je “Allow always” gebruiken voor de dingen die je vertrouwt.
4

Superpowers — Brainstormen met Claude

Dit is de kern van deze handleiding, vriend. Superpowers zijn slimme werkwijzen die Claude extra gestructureerd laten werken. In plaats van zomaar code te laten schrijven, laat je Claude eerst meedenken over je idee.

Superpowers installeren (eenmalig)

Superpowers zitten niet standaard in Claude — je installeert ze via de marketplace. Dat doe je één keer:

Claude Code
> /market

Er opent een overzicht van beschikbare uitbreidingen. Zoek naar superpowers en installeer het. Klaar! Dit hoef je maar één keer te doen.

De brainstorm starten

Nu kun je brainstormen. Typ het volgende, gevolgd door je idee in gewone taal:

Claude Code
> /superpowers:brainstorming
Ik wil een website maken waarop ik mijn favoriete recepten kan delen

Hoe het werkt

Claude gaat nu iets slims doen: in plaats van meteen code te schrijven, stelt Claude vragen. Eén voor één. Vaak meerkeuze, zodat je makkelijk kunt kiezen.

Claude's brainstorm-vragen (voorbeeld)
Leuk idee! Laten we het uitwerken.

Wat voor soort recepten wil je delen?
  a) Alle soorten door elkaar
  b) Een specifieke keuken (Italiaans, Aziatisch, etc.)
  c) Een specifiek thema (budget, snel klaar, vegetarisch)

> a


Wil je dat bezoekers ook recepten kunnen toevoegen,
of ben jij de enige die recepten plaatst?
  a) Alleen ik
  b) Bezoekers mogen ook recepten toevoegen

> a


Heb je al foto's van je gerechten?
  a) Ja, die staan in mijn werkmap
  b) Nee, gebruik placeholder-afbeeldingen

> a, ze staan in de map foto's

Jij kiest, Claude denkt mee. Zo bouw je samen een helder beeld van wat je wilt.

Tip: Heb je referentiemateriaal in je werkmap gezet — een screenshot van een site die je mooi vindt, een Word-doc met teksten, foto's? Verwijs ernaar! Zeg bijvoorbeeld: “Ik heb een screenshot van een site die ik mooi vind in mijn map gezet, gebruik die als inspiratie.”

Het resultaat: jouw blauwdruk

Na de brainstorm maakt Claude een samenvatting van alle keuzes: het ontwerp, de structuur, de kleuren, alles. Dit wordt opgeslagen als jouw blauwdruk.

Claude vraagt: “Ziet dit er goed uit? Wil je iets aanpassen?” Pas het aan totdat je tevreden bent. Dan gaan we bouwen!

5

Van plan naar werkende site (en meer)

Na de brainstorm gaat het snel. Claude maakt automatisch een bouwplan en gaat stap voor stap aan de slag.

Claude bouwt, jij kijkt mee

Je ziet bestanden verschijnen in je werkmap. Claude vertelt je precies wat hij doet:

Claude aan het werk
✓ Bestand aangemaakt: index.html
✓ Bestand aangemaakt: style.css
✓ Afbeeldingen verwerkt: 6 foto's geoptimaliseerd
✓ Recept-pagina's aangemaakt: 12 recepten

Je site is klaar! Open index.html in je browser
om het resultaat te bekijken.

Tussendoor bekijken

Open index.html in je browser (dubbelklik op het bestand). Niet helemaal wat je in gedachten had? Geen probleem!

Aanpassingen vragen
> De kleuren zijn te donker, maak het wat vrolijker
> Kun je de foto's groter maken?
> Voeg een zoekbalk toe bovenaan

Aanpassen is normaal. Vibecoding is een dialoog — je bouwt samen met Claude, net zo lang tot het goed is.

Meer dan alleen websites!

Claude kan veel meer dan websites bouwen. Alles wat in je werkmap staat, kan Claude ermee werken:

  • Een Excel analyseren — “Analyseer de verkoopcijfers in mijn Excel en maak er grafieken van”
  • Een PowerPoint maken — “Maak een presentatie van 10 slides over dit onderwerp”
  • Een rapport samenvatten — “Vat dit PDF-rapport samen in 5 kernpunten”
  • Data omzetten — “Zet deze CSV om naar een overzichtelijke tabel”
Onthoud: Alles wat in je werkmap staat, kan Claude zien en gebruiken. Zet er dus alles in wat je nodig hebt!
6

Live zetten op GitHub Pages

Je hebt een werkende site. Maar nu wil je hem delen met de wereld! Dat kan gratis via GitHub Pages.

Wat is GitHub Pages?

Een gratis dienst van GitHub waarmee je een website online zet. Je krijgt een adres zoals:

Jouw website-adres
jouwgebruikersnaam.github.io/mijn-recepten

Dat adres kun je delen met iedereen — familie, vrienden, collega's.

Claude doet het werk

Het mooie? Je hoeft niet te weten hoe GitHub werkt. Zeg gewoon:

Claude Code
> Zet dit live op GitHub Pages

Claude regelt alles:

  1. Claude vraagt eenmalig om in te loggen bij GitHub (via je browser)
  2. Claude maakt een repository aan (een soort online map)
  3. Claude uploadt je bestanden
  4. Claude activeert GitHub Pages

Binnen een paar minuten staat je site live! Claude geeft je de link.

Later aanpassen?

Start Claude opnieuw in dezelfde werkmap en zeg wat je wilt wijzigen. Claude past het aan en zet de nieuwe versie online.

Aanpassing doorvoeren
> cd C:\mijn-project   (of ~/mijn-project op Mac)
> claude
> Voeg een nieuw recept toe voor pasta carbonara
> Zet de update live op GitHub Pages

En nu zelf — Cheatsheet

Je weet nu alles wat je nodig hebt, vriend! Hier is je spiekbriefje voor als je aan de slag gaat:

Eenmalig (heb je al gedaan)

Claude Code geïnstalleerd + tokens gekocht
GitHub account aangemaakt
Superpowers geïnstalleerd via /market

Elk nieuw project

1. Map aanmaken met je materiaal
2. Terminal openen, naar de map, claude typen
3. /superpowers:brainstorming + je idee
4. Vragen beantwoorden, ontwerp goedkeuren, laten bouwen
5. "Zet het live op GitHub Pages"

Handige zinnen voor Claude

"Gebruik de bestanden in mijn map" "Ik wil het anders, kun je..." "Maak er een PowerPoint van" "Analyseer deze Excel" "Zet dit live op GitHub Pages" "Maak de kleuren wat warmer" "Voeg een zoekfunctie toe"

Problemen?

  • Claude doet niks? Check of je in de juiste map staat. Typ pwd (Mac) of cd (Windows) om te zien waar je bent.
  • Iets mislukt? Gewoon zeggen wat er fout ging. Claude probeert het opnieuw. Vibecoding is proberen, bijsturen, verbeteren.
  • Tokens op? Koop een nieuw bundeltje via je Claude-account.
Veel plezier met vibecoden, vriend!
